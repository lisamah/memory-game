# 🃏 Memory Game

A classic card-matching memory game built with TypeScript, HTML, and CSS.

## Description

Memory Game is a browser-based card game where the goal is to find all matching pairs of cards. Cards are shuffled on each game start, so every round is a new challenge. Once all pairs are found, a congratulations screen appears and you can start a new round.

## Technologies

- **TypeScript** – Game logic and DOM interaction
- **HTML5** – Structure and markup
- **CSS3** – Styling and layout
- **npm** – Build tooling

## Installation & Setup

### Prerequisites

- [Node.js](https://nodejs.org/) installed

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/lisamah/memory-game.git
   cd memory-game
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the TypeScript files:
   ```bash
   npm run build
   ```

4. Open `index.html` in your browser and start playing!

### Development

To automatically recompile TypeScript on file changes, use the watch mode:
```bash
npm run watch
```
