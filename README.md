# Memory Game

Ein einfaches Memory-Spiel, das mit TypeScript und HTML/CSS gebaut wurde.

## Features

- 12 Karten (6 Paare) zum Aufdecken
- Zufälliges Mischen der Karten
- Automatisches Zurückdrehen bei nicht-passenden Paaren

## Installation

```bash
npm install
```

## Entwicklung

TypeScript compilieren:
```bash
npm run build
```

Oder im Watch-Mode:
```bash
npm run watch
```

Dann `index.html` im Browser öffnen.

## Spielweise

1. Klicke auf die Karten, um sie aufzudecken
2. Versuche, passende Paare zu finden
3. Wenn zwei Karten nicht passen, werden sie nach 1 Sekunde wieder verdeckt
